About

How I think about building hardware that actually ships.

I’m an embedded systems and product design engineer who enjoys owning the full stack of a hardware product—from the first block diagram to the final enclosure on someone’s desk.

On the electronics side, I work on mixed‑signal PCBs, power stages, and microcontroller‑based systems. I care about layout discipline, signal integrity, and designing boards that are easy to test, debug, and scale into production.

On the firmware side, I write C/C++ for MCUs, build real‑time control logic, and integrate connectivity for IoT devices. I like firmware that is predictable, well‑structured, and documented enough that another engineer can pick it up quickly.

For mechanical design, I model enclosures and assemblies that respect real‑world constraints: tolerances, fasteners, cable routing, and how a technician will actually assemble or service the device.

What I bring

  • End‑to‑end hardware product development
  • Mixed‑signal PCB and power design
  • Firmware for real‑time and IoT systems
  • Mechanical design for enclosures and assemblies

Embedded Audio System

Embedded Audio

The system is built around an ESP32-S3 microcontroller, integrating a digital I2S microphone for capture and Class D amplifier for playback. Audio is stored on SD card in standard WAV format for universal compatibility.

  • One-Button Recording - Press START to record, STOP to finish
  • Instant Playback - Play last recorded audio with single button press
  • WAV Format - Universal file format, compatible with all audio software
  • Auto File Numbering - Automatic sequential naming (rec_1.wav, rec_2.wav, etc.)
  • Up to 60 Seconds Recording - Configurable maximum recording time per file
  • Auto Power Switching - Seamlessly switches between USB and battery
  • Smart Amplifier Control - Automatic power-down when not playing (saves battery)
  • Onboard Antenna - Chip antenna with proper RF design
ESP32 • RF • KiCad Signal integrity • Power design

AI Timer system

Smart Home

This project is a programmable timer module designed for automation and control applications. It combines practical electronic design with simple user interaction, making it suitable for controlling 220V AC loads based on set time intervals.

  • Manual Time Configuration -Equipped with four push buttons — SET, UP, DOWN, and START — for easy time adjustment and operation.
  • Display Interface -An I²C port is provided for connecting an external display to show real-time timing data and system status
  • Programming Interface -UART pins are available for uploading firmware and serial communication.
  • Load Control -Supports 220V AC load automation, allowing direct control of external devices.
  • Real-Time Feedback -Integrated LED indicators (Power, Alert, Active) and a buzzer provide clear system feedback during operation.

Air Quality Monitoring system

IoT & Sensing

The Quality Air Monitoring System is a compact environmental sensing device designed to track and display key air quality parameters in real time. It combines precise sensing, onboard data processing, and a user-friendly interface to help users monitor and understand their surrounding air conditions.

  • USB to UART Interface - Simplifies firmware updates and data communication with external devices.
  • In-Built Battery Charger - Supports rechargeable power for portable or continuous operation.
  • I²C Display Port - Allows easy integration of external displays for real-time readings.
  • Real-Time Feedback - Includes a buzzer for instant audible alerts on poor air conditions or threshold triggers.
  • Real-Time Clock (RTC) - Keeps accurate time for timestamped readings and event logging.

6 channel smart switch

Smart Home

The 6 Channel Smart Switch is a multi-output control board designed for home automation, energy management, and small-scale industrial control systems. It allows users to manage multiple devices or loads from a single compact unit.

  • 6 Independent SSR Relay Channels
    • -Enables switching of multiple loads safely and efficiently.
  • Button Connector - Allows connection of up to six external switches for manual control.
  • Temperature Sensor Port - Supports external temperature monitoring for automated responses.
  • Extra Power Pins -Includes additional 3.3V and 5V outputs for powering sensors or modules.
  • Built-in RC Snubber Protection -Improves relay durability and safeguards the system against voltage spikes.
